Description of feature filling

All the first feature that were unknown were just replaced by b

For the feature that are not numbers i will find the median value between all the possible value of the feature

for the real-valued feature i will find the label-conditioned mean (i.e., Î¼(x1|+)) which will for the + data
(Sum of all the particular feature for + data set )/ number of + data sets

For the - data, it will be

(Sum of all the particular feature for - data set )/ number of + data sets

for the attribute features we need to replace by the mode of all the attributes
